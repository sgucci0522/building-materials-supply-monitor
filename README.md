# Building Materials Supply Monitor

中東情勢起因の建材供給制約をモニタリングし、供給回復のサインを Slack に即時通知するシステム。

## 背景

2026年4月時点で、ナフサショックにより以下の建材で供給制約・受注停止・大幅値上げが発生:

- 塩ビ管 (クボタケミックス・積水化学・アロン化成 等)
- FRP用不飽和ポリエステル樹脂 (ユピカ4190 等)
- ガラスマット
- 防水材 (田島ルーフィング オルタック 等)
- 塗料・シンナー・シーリング材

これらが「いつ供給再開されるか」をリアルタイムに把握するためのツール。

## 仕組み

```
[GitHub Actions (平日 7時/12時 JST)]
        ↓
[Python: 各メーカーサイトをスクレイピング]
        ↓
[前回スナップショットとdiff]
        ↓
[キーワード抽出: 再開🟢 / 制限🔴 / 関連🟡]
        ↓
[Slack Webhook で通知]
```
